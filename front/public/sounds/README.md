# Sons do Jogo

Coloque arquivos MP3 aqui para substituir os sons sintéticos gerados pela Web Audio API.

Arquivos esperados:
- `splash.mp3` — Tiro na água (miss)
- `explosion.mp3` — Acerto em navio (hit)
- `sunk.mp3` — Navio afundado
- `victory.mp3` — Vitória (fanfarra)
- `defeat.mp3` — Derrota (tambores graves)
- `click.mp3` — Click em botões
- `ambient.mp3` — Música de fundo pirata (loop)

Se o arquivo não existir, o jogo usa som sintético gerado pela Web Audio API.
